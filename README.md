# Cotizador Simple para Servicios Informales

Versión mejorada visualmente, lista para Railway.

## Cómo correr

```bash
npm install
npm start
```

Abre:

```text
http://localhost:3000
```

## Datos

Esta versión guarda cotizaciones temporalmente en:

```text
data/quotes.json
```

Para producción se recomienda migrar a PostgreSQL en Railway.
